# EmbyBoss 服务使用指南

这是关于名为 `EmbyBoss` 的 Telegram 机器人的功能使用手册。
如果你在群里被问到如何管理 Emby 账号、获得积分或者相关服务，请参考以下功能指令进行回答。

## 普通用户指令

- `/start` : 在私聊中使用，唤醒机器人并展示基础控制面板。如果是在群组中使用会提醒私聊。
- `/myinfo` : 获取你的个人状态信息、注册状态、积分余额和 Emby 权限信息。
- `/count` : 查询当前服务器的影视媒体数量。
- `/exchange` : 凭借积分或激活码兑换相关权益。
- `/bet` 或 `/startbet` : 启动或者参与积分竞猜游戏。
- `/win` / `胜率` : 查看个人的游戏胜率。
- `/wintop` / `胜率榜` : 调出群内玩家胜率排行榜。
- `/g21` : 玩 21 点小游戏，以赢取积分。

## 管理员指令 (仅管理员或 Owner 可用)

- `/kk <TG ID>` 或对某人讲话回复 `/kk` : 核心管理与装 X 指令！调出该成员的数据面板，可以用按钮直观操作：**赠送入场资格**、**封禁/解封账号**、**开关额外媒体库(隐藏库)**、**直接删号**或**直接将其从群聊中踢出并封禁**。
- `/config` : 呼出管理员配置面板。
- `/check_ex` : 检查所有用户的活跃度与过期状态。
- `/low_activity` : 查询或设定低活跃成员踢出阈值。
- `/backup_db` : 备份当前数据库资料。
- `/days_ranks` / `/week_ranks` : 导出日活跃、周活跃的群内用户积分排行。
- `/uranks` : 总积分排行管理。
- `/sync_favorites` : 同步用户的收藏夹数据。
- `/restart` / `/update_bot` : 重新启动机器人或从源码拉取最新版本更新。
- `/score` / `/coins` : 手动为特定用户补录或扣除积分。
- `/syncgroupm` / `/syncunbound` : 同步群组成员、清理未绑定 TG 账号的 Emby 用户。
- `/bindall_id` : 批量帮所有已有账号绑定 TG ID。
- `/embyadmin` : 为指定用户提升为 Emby 系统管理员权限。
- `/deleted` / `/rmemby` : 强行删除指定违规用户的 Emby 账号和群内记录。
- `/kick_not_emby` : 踢出没有开通或者绑定 Emby 的低活跃/闲杂群组成员。
- `/paolu` : 清空或销毁核心数据。
- `/rm_disabled` : 一键将所有显示处于禁用状态的账号删除。
- `/banall` / `/unbanall` : 批量封禁/解封用户。
- `/renewall` / `/coinsall` : 批量为所有人续期、发放积分补偿等普惠操作。

## 常见操作 Q&A
**Q: 获取或找回密码怎么办？**
**A:** 请私聊机器人发送 `/start` 或者点击菜单进入个人面板进行找回。

**Q: 我的号被删除了，或者没有分数了怎么办？**
**A:** 请在群内保持活跃获取积分，或通过机器人面板重新获取入场券。如果是被误删，请通过 `/myinfo` 提供截图并联系管理员。
